# AppStore-APIs
An AppStore clone application using various JSON APIs coming directly from Apple's iTunes Search API. 

<strong>Technologies used:</strong><br><br>
• No storyboards<br>
• MVVM<br>
• Decodable<br>
• Cocoapods<br>
• Nested UICollectionViews<br>
• SDWebImage
